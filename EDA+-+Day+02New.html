Boxplot¶

  • Boxplot is a numerical summary of the data that consists of Min, Max, 25th Percentile, Median (50th) and 75th Percentile data

  • We can identify the Outliers basis the Boxplot

  • We calculate the Inter Quartile Range (IQR) => 75-25
  • Lower Limit = Q1-1.5*IQR
  • Upper Limit = Q3+1.5*IQR

Note - Any value which is less than Lower Limit or Greater than Upper Limit is considered to be an Outlier

Categorical variable Encoding¶

  • One Hot Encoding is a form of Encoding where we treat all the categorical values as same and an identitiy matrix is generated to ensure that each category in the column gets equal weightage. For e.g. pd.get_dummies()

  • Label Encoder - It is applied on those cat variables where we see some order such as one is greater than the other. In our data, Outlet Size is a variable which has a certain order (Small, Medium & Large).

  • Therein we will apply Label Encoder to Encode the values as 1,2,3... so that machine can understand that one category has more weightage than the other... This is achieved by Label Encoding...
